# Capacité : Création de Présentations Marp

## 1. Description
L'agent est capable de concevoir des diaporamas au format Markdown pour Marp.

## 2. Standards et Règles
- **Frontmatter** : Ajout systématique du frontmatter YAML (`marp: true`, `theme`, `paginate: true`).
- **Séparation** : Séparation des diapositives par `---`.
- **Structure** : Structuration aérée : 1 idée claire par slide.
