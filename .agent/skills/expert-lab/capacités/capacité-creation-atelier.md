# Capacité : Création d'Atelier

## 1. Description
L'agent est capable de concevoir et rédiger des tutoriels d'ateliers pédagogiques.

## 2. Standards et Règles
- **Pratique** : L'atelier guide l'apprenant pas à pas pour réaliser une tâche technique précise.
- **Clarté** : Fournir les commandes et le code nécessaires ou guider l'apprenant vers la solution.
- **Objectif** : Chaque atelier valide un acquis de la présentation associée.
